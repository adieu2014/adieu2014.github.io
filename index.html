Resource Handling in
Spring MVC

Spring MVC에서 정적 자원(css, js, etc)을 다루는 방법을 살펴본다.

Frontend Module과 Backend Module로 구성된 웹 애플리케이션을 개발한 사례를 공유한다.
Frontend는 grunt, bower 등의 도구를 기반으로 서버에서 사용할 정적 자원(html, css, js, etc)으로 구성되었고, Backend는 gradle, spring boot & spring mvc를 기반으로 개발되었다.
Frontend Module이 제공하는 자원을 Backend Module에서 어떻게 다루는지에 대해 중점을 두고 설명한다.

프론트엔드 개발프로세스
어디까지 개선할 수 있나

프론트엔드 개발 프로세스의 생산성을 높히는 방법을 설명합니다.

최근 몇 년 사이에 프론트엔드 개발과 관련된 도구가 많이 등장하고 개발 프로세스가 급격하게 개선되고 있습니다. Grunt, gulp, Bower 등의 좋은 도구들이 많이 있고 사용방법도 어려운 편은 아니지만, 도구도 너무 많이 존재하고 실제 프로젝트에 어떤 식으로 적용해서 사용해야 하는지는 파악하기가 쉽지 않습니다.

각 도구의 사용방법 보다는 프로젝트의 개발 프로세스상에 어떤 식으로 적용해서 생산성을 개선할 수 있는지를 설명합니다.

스위프트 프로그래밍

Swift로 앱 개발하기

약 50분간 스위프트와 XCode를 사용해서 간단한 ToDo 관리 앱 개발을 시연합니다. 개발을 시연하면서 스위프트 프로그래밍 언어의 특징과 문법을 설명하며, 마지막엔 동일한 앱을 Objective-C로 개발한 코드를 보며 비교합니다.

스위프트로 개발한 앱의 소스 코드는 여기서 확인할 수 있으며, Objective-C로 개발한 앱의 소스 코드는 여기서 확인할 수 있습니다.

Building REST API Server

Github과 비슷한 API 서버를 만들어 보자. 같이.

RESTful 웹 서비스를 개발중인데, 응답의 Status Code는 어떻게 보낼지, 인증 처리는 어떻게 할 것인지, 예외 처리는 어떻게 할것이며, JPA 연동 등의 고민을 해왔고 나름의 방법을 찾아 개발 중입니다.

자바8 기반이고 프레임워크로는 스프링 붓(boot), 스프링 데이터 JPA, 스프링 시큐리티를 사용중이며 그밖에도 ModelMapper, JacksonJson, Lombok 등의 라이브러리를 사용하고 있습니다.

노트북에 자바 8과 자신에게 익숙한 통합 개발 환경(이클립스 또는 인텔리J)을 준비해 오신다면 현장에서 개발하는 과정을 실시간으로 따라하실 수 있습니다.
